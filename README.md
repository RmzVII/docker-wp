## 1. Встановлення WSL2
1. Відкрити PowerShell від імені адміністратора та виконати:
wsl –install
2. Перезавантажити комп’ютер.
3. Клонування репозиторію
Відкрити Ubuntu і виконати:
git clone https://github.com/RmzVII/docker-wp ~/wpmanager
cd ~/wpmanager
________________________________________
3. Встановлення WordPress Manager. виконати команду:
bash install.sh
echo $PATH
o	create_wp — створює новий сайт
o	run — керує конкретним сайтом (start/stop/logs/open)
o	wpmanager — меню для керування сайтами
⚠ Якщо це перший запуск — потрібно перезапустити WSL:
wsl --shutdown
________________________________________
4. Створення нового сайту
create_wp <назва_сайту> <порт>
Приклад:
create_wp school_project_1 8081
Сайт буде доступний: http://localhost:8081
________________________________________
5. Керування сайтом
•	Запуск сайту:
run school_project_1 start
•	Зупинка сайту:
run school_project_1 stop
•	Перегляд логів:
run school_project_1 logs
•	Відкрити сайт у браузері:
run school_project_1 open
________________________________________
6. Менеджер сайтів з меню
wpmanager
Меню дозволяє:
1.	Створити новий сайт
2.	Запустити сайт
3.	Зупинити сайт
4.	Видалити сайт
5.	Подивитися список сайтів
6.	Статус Docker
7.	Очистити Docker (видалити всі контейнери)
8.	Вихід
Зручно для учнів, щоб працювати без командного рядка.
________________________________________
7. Поради
•	Кожен сайт ізольований у ~/projects/<назва_сайту>
•	Використовуй різні порти для кожного сайту
•	Для швидкого керування — wpmanager або run <сайт> start/stop
•	Якщо Docker не працює — перевір systemd у WSL та перезапусти:
wsl --shutdown
